# Login Images Directory

## Hero Image

Place your login hero image here with the filename: `login-hero.png` (or `.jpg`, `.webp`)

The image should be:
- High resolution (recommended: 1920x1080 or higher)
- Format: JPG, PNG, or WebP
- Optimized for web (compressed but high quality)
- Shows a professional woman with holographic interface

The image will be displayed on the left side of all login pages.

### Current Expected File:
- `/public/login/login-hero.png`

This image is accessible at: `/login/login-hero.png` in your application.

If you have the image with a different name or format, you can either:
1. Rename it to `login-hero.png` (or keep your format: `.jpg`, `.webp`, etc.)
2. Update the `heroImageSrc` prop in the login pages (e.g., `app/console/login/page.tsx`)
