# Application Configuration

This folder contains application-level configuration files (not tool configs).

## Structure

```
config/
 └─ api.config.ts    # API endpoint configuration
```

## Note

Tool-specific configuration files (Next.js, TypeScript, Tailwind, etc.) are in the root directory as required by those tools, with organized copies in `.config/` folder.

For tool configs, see `.config/README.md`.
