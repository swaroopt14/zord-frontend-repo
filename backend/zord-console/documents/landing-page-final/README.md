# Landing Page Final

This folder is the clean placeholder/reference location for the final landing page work.

The actual landing page implementation lives in:

- `app/page.tsx`
- `components/landing/`
- `app/globals.css`

Useful entry points:

- Root route: `/`
- Main page file: `app/page.tsx`
- Hero and section components: `components/landing/`

Reason for this folder:

- keeps landing-page references out of the repo root
- gives the landing work a stable final name: `landing-page-final`
